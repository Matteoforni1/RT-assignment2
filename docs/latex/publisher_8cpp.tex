Description of the \mbox{\hyperlink{publisher_8cpp_a47e9ad161b3ce5263fd2f8005045d9b6}{my\+Callback()}} function\+:

This function takes an Odometry message as parameter

It does not return anything

This function is the callback function for the subscriber to the topic \textquotesingle{}/odom\textquotesingle{}. This function retrieves from the Odometry message the current robot position and velocity (both linear and angular). Then we create an info\+\_\+robot message, where we put the information mentioned above and then we publish this message on the topic \textquotesingle{}/my\+\_\+info\textquotesingle{}.
