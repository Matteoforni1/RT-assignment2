Description \+:

This node is the action client and user interface for the second assignment of the course Research Track 1. This Node implements a user interface through a menù which is printed on the terminal. The user is aked to type one of four commands\+: to set a new goal, to cancel the current one, to print the number of reached and cancelled goals and to terminate the execution.

This function does not take any parameter

It does not return anything

This function prints a menù, where the user is asked to insert a command from keyboard. If the user presses \textquotesingle{}g\textquotesingle{}, then the coordinates for a new goal will be asked; once entered, this node (as an action client) send the goal coordinates to the action server node. If the user presses \textquotesingle{}c\textquotesingle{}, we cancel the current action, which is the pursuit of a goal. In both cases, it is checked that when we set a goal, no goal is currently active and when we cancel a goal, there is actually a goal to cancel. If the user presses \textquotesingle{}p\textquotesingle{}, we use the service node to retrieve the number of goal reached and cancelled and then we print them. If the user presses \textquotesingle{}e\textquotesingle{}, we terminate the execution. \mbox{\Hypertarget{action__client_8cpp_a9ffe2fced00dd42867bb463a7f5f177f}\label{action__client_8cpp_a9ffe2fced00dd42867bb463a7f5f177f}}
